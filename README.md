# Customer Churn Analysis (Python)

## Overview
This project analyzes customer churn behavior using Python to identify key factors contributing to customer attrition.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Findings
- Month-to-month contracts have significantly higher churn
- Higher monthly charges increase churn risk
- Long-tenure customers are more likely to stay

## Business Value
The analysis provides insights that can support customer retention strategies, pricing decisions, and contract optimization.

## Files
- `customer.ipynb`: Full analysis notebook
- `telco_churn.csv`: Dataset
